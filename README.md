# Avito: Детекция запрещенных товаров (CV)

## О задаче

Разработка модели компьютерного зрения для автоматической модерации объявлений на Avito. 
Задача — классифицировать изображения и выявлять курительные принадлежности (кальяны, вейпы), размещение которых запрещено правилами площадки.

**Результат:** ROC-AUC 0.9881 на приватном лидерборде.

## Используемые библиотеки и технологии

В проекте задействован современный стек для Computer Vision:

- **PyTorch & Torchvision** — основной фреймворк

- **Albumentations** — библиотека для сложных аугментаций изображений

- **EfficientNet (B4)** — архитектура сети (было важно качество картинки, поэтому остановилась на этой архитектуре)

- **OpenCV** — предобработка изображений, ресайз и чтение файлов

- **Scikit-learn** — реализация Stratified K-Fold валидации и подсчет метрик

- **Pandas & NumPy**

## Особенности решения

- Использование Mixup аугментации для улучшения обобщающей способности

- Двухэтапное обучение: Warmup (сначала обучаем только голову отдельно) -> Fine-tuning (потом дообучиваем всю сеть, размораживая все веса)

- Взвешенная функция потерь (`BCEWithLogitsLoss`) для работы с дисбалансом классов

**Валидация и инференс:**

- Обучение на 3 фолдах (Stratified K-Fold), данных было мало, так что это было оптимальным решением

- TTA (Test Time Augmentation): Усреднение предсказаний для оригинального изображения и его зеркальной копии

- Ансамблирование моделей с разных фолдов
